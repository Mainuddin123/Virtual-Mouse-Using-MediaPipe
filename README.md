# 🖱 AI Virtual Mouse Using MediaPipe

## Overview

AI Virtual Mouse is a Computer Vision project that allows users to control mouse operations using hand gestures captured through a webcam.

The system uses MediaPipe Hand Tracking and OpenCV to detect hand landmarks and perform mouse actions such as cursor movement, clicking, dragging, and scrolling.

---

## Features

* Real-Time Hand Tracking
* Cursor Movement
* Left Click Gesture
* Right Click Gesture
* Drag and Drop
* Scroll Control
* Frame Capture
* Video Recording
* Streamlit Dashboard

---

## Technologies Used

* Python
* OpenCV
* MediaPipe
* PyAutoGUI
* NumPy
* Streamlit

---

## Workflow

Webcam → Hand Detection → Landmark Extraction → Gesture Recognition → Mouse Control
